The JotLinks Like Button adds a 'Like' button to your Wordpress blog posts.

== Description ==
Let your readers quickly save and share your content on JotLinks with a simple click!

You can customize it in the Settings section:

* Layout (jlbtnlarge, jlbtnsmall, jlbtnmini or jlbtnicon)
* Align to the Left or Right of your posts
* Show at the top and/or bottom of posts
* Show/hide the button on pages/posts/home/search/archive.
* Margins (top, bottom, left, right)
* Support of the Open Graph protocol (http://opengraphprotocol.org)

The plugin configures the Open Graph Meta tags you need in your HTML header:

* og:site_name
* og:title
* og:type
* og:description
* og:url

Visit the <a
href="http://www.jotlinks.com/developers">JotLinks Plugin Homepage</a> for more information.

== Installation ==

1. Download the latest version (ex: jotlinks-button.zip)
2. Extract it in the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. (Optional) Customize the plugin in the Settings > JotLinks menu


Visit the <a
href="http://www.jotlinks.com/developers">JotLinks Plugin Homepage</a> for more information.
